## 🚀 上线部署步骤

### 1. 环境配置 (即刻执行)
```bash
# 后端环境配置
cd runninghub-backend
cp .env.production.template .env
# 配置微信支付和支付宝凭据

# 数据库迁移
npm run db:push

# 启动生产环境
npm start
```

### 2. 前端部署 (Vercel)
```bash
cd project
# 构建生产版本
npm run build

# 部署到Vercel
vercel deploy --prod
```

### 3. 监控启动
```bash
# 启动监控服务
docker-compose up -d
```
