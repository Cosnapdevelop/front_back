🎯 创建Cosnap测试账号

📋 使用说明：

1. 输入一个邮箱地址作为测试账号

2. 选择VIP等级（推荐）- 获得无限AI特效使用权限

3. 点击创建按钮

4. 创建成功后，使用该邮箱登录前端网站即可

可以使用任何邮箱地址，不需要真实存在

🔧 如果遇到CORS错误，请使用以下备用方法：

方法1 - 使用curl命令（推荐）:

curl -X POST "https://cosnap-back.onrender.com/api/admin/test-user" -H "x-admin-key: cosnap-test-admin-2024-secure-key" -H "Content-Type: application/json" -d "{\"email\": \"test@cosnap.dev\", \"tier\": \"VIP\"}"

方法2 - 使用Chrome浏览器（禁用安全检查）:

1. 关闭所有Chrome窗口

2. 按Win+R，输入以下命令：

chrome --disable-web-security --user-data-dir="C:\temp\chrome_dev"

3. 用新打开的Chrome重新访问这个页面

⚠️ 此工具仅用于测试目的
